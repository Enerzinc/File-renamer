# File Renamer Script

This is a simple Python Script that renames all files inside a specified folder by removing dashes (-) from their filenames.
The program is created using Object-Oriented Programming (OOP) principles and uses the `pathlib` library for handling file paths.

Example:

Before:
12-121-241.txt  
my-file-name.docx  

After:
12121241.txt  
myfilename.docx  

---

##  Technologies Used

- Python 
- pathlib library
  
---

## How to Run the Program (Using CMD in Windows)

### Step 1: Check if Python is Installed

Open Command Prompt and type:

python --version

If it shows a Python version, you are ready.

### Step 2: Open CMD in the Folder Where the File Is Saved

1. Open the folder where your Python file is saved.
2. Click the address bar.
3. Type: cmd
4. Press Enter.

### Step 3: Run the Program

Type:
python main.py
Then press Enter.
